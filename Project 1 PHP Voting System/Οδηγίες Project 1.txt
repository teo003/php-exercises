Project 1 - Voting System with PHP
1)Στον φάκελο htdocs του XAMPP δημιουργούμε τον φάκελο php-voting-system
2)Αντιγράφουμε εντός αυτού τα έτοιμα αρχεία.
3)Ξεκινάμε τα Apache και MySQL στο XAMPP και κάνουμε κλικ στο δεύτερο Admin.
4)Δημιουργούμε την νέα βάση δεδομένων polltest στην οποία και αντιγράφουμε το στοιχείο εντός του φάκελου db που δόθηκε.
5)Γράφουμε τα αρχεία index.php, header.php, footer.php, submit_vote.php, voter.php
6)Στον browser της επιλογής μας γράφουμε localhost/php-voting-system
7)Για να ψηφίσουμε πρέπει πρώτα να κάνουμε εγγραφή (Register) και έπειτα σύνδεση (login) με το username και το password που δώσαμε.
8)Τέλος μπορούμε να δούμε τα αποτελέσματα με βάση ότι υπάρχει στην βάση δεδομένων που χρησιμοποιήσαμε.
